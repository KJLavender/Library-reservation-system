bbb資料夾為PHP程式碼，我是用XAMPP開啟phpMyadmin的。而且bbb資料夾必須放在XAMPP中的htdocs。

下載完XAMPP後，打開他，會發現名稱那邊有Apache跟MySQL，點選這兩個的Start還有MySQL的admin。

接著就會帶你到phpMyadmin的網頁，進去之後，bbb(1)為.sql檔，按匯入選擇bbb(1).sql再匯入就好。

先在另一個網頁上打上 : localhost開啟網頁伺服器(http://localhost/dashboard/)。

再新增一個新網頁並打上 : localhost/bbb/new.php，就會有我們做的網頁了。
